🎮 游戏操作

  • 键盘：方向键 ↑↓←→ 或 WASD
  • 手机：滑动屏幕 或 点击方向键
  • 暂停：按 ESC 或点击暂停按钮

🐍 游戏规则

  • 控制蛇吃到食物（红色方块 +1分）
  • 金色食物 +3分，彩虹食物 +5分
  • 撞到墙壁或自己的身体，游戏结束
  • 简单模式下可以穿墙（从对面出现）

🏆 游戏模式

  • 经典模式：无限时，追求最高分
  • 限时模式：在规定时间内获得尽可能高的分数
  • 障碍模式：地图中有障碍物，小心避开

🎖️ 成就系统

  • 完成特定目标可解锁成就
  • 在菜单中查看所有成就
  • 继续挑战，解锁全部成就吧！
