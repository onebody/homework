- 所有类/模块文件末尾统一使用 `if (typeof module !== 'undefined' && module.exports) { module.exports = ... }` 同时支持浏览器全局与 Node require。
- 游戏配置（网格尺寸、难度、模式、食物类型、皮肤、默认设置、localStorage key）集中在 `Constants.js` 中以大写常量对象导出，运行时直接读取而非传参。
- 跨组件通信一律通过全局 `globalEventBus` 发布 `game:*` / `audio:play` 等带命名空间的事件名，避免直接相互引用。
- 碰撞检测采用纯函数式对象 `Collision`，仅接收坐标与边界参数，不持有状态，由 `GameEngine.update` 组合调用 `checkAll` 后统一处理结果。
- 移动输入使用 `setDirection` 写入 `nextDirection`，在每帧 `move` 开始时才同步到 `direction`，以此防止同一帧内 180° 反向转向。
- 视觉效果（粒子、飘字、死亡闪烁）以数组+生命周期衰减字段（life/decay/vy）管理，在 `updateEffects` 中 filter 清理并在 `render` 中绘制，与逻辑更新分离。
