# 用户操作手册

> 暑假作业打卡系统（三年级）· 面向学生、家长、管理员
> 版本：v1.0

---

## 一、学生操作指南

### 1. 注册与登录
1. 手机浏览器打开学生端地址（如 `http://<服务器>/`）。
2. 点击「注册」，选择「我是学生」，填写用户名、昵称、密码，提交进入。
3. 进入「我的」页面可看到你的**绑定码**（如 S00001），发给家长用于绑定。

### 2. 采集人脸底图（推荐，防代打卡）
1. 进入底部「👤 我的」→「😊 人脸打卡验证」。
2. 点击拍摄/选择一张清晰正脸照，点「采集人脸底图」。
3. 采集后，每次打卡会自动比对是否本人。可随时「撤销人脸底图」。

### 3. 每日打卡（3 步完成）
1. 底部点「📝 打卡闯关」，默认在「📸 每日打卡」子页。
2. **步骤 1**：点击上传作业现场照片。
3. **步骤 2**（可选）：若补过去某天的卡，勾选「这是补卡」，选日期、填原因、传凭证。
4. **步骤 3**：点「获取位置」，再点「完成打卡」。正常打卡 +10 积分，补卡 +5 积分。
5. 打卡提交后进入待审核，管理员通过后积分到账、连续天数增加。

### 4. 闯关任务
1. 在「📝 打卡闯关」页点上方「🏆 闯关任务」子 tab。
2. 点任务查看详情，填写完成说明 + 上传完成照片，提交打卡。
3. 在「我的闯关打卡记录」查看审核状态。

### 5. 抽奖
- 连续打卡每满 7 天获得 1 张抽奖券（永久累积）。
- 底部点「🎰 抽奖」，查看券数量，点转盘中心「抽」或下方「🎰 抽一次」。
- 转盘转动后显示中奖结果；中奖记录可在下方查看。

### 6. 积分商城
- 底部点「🛍️ 商城」查看积分余额与可兑换奖品。
- 点「兑换」用积分换实物奖品；也可兑换「抽奖券」增加抽奖机会。
- 「我的兑换」里可对已兑换奖品做「替换」。

### 7. 查看学习报告
- 「👤 我的」→「📊 查看 / 下载学习报告」，查看暑假打卡频率、最长连续、完成情况，可打印下载。

### 8. 修改密码
- 「👤 我的」→「🔑 密码设置」→ 输入旧密码与新密码确认修改。

## 二、家长操作指南

### 1. 注册与绑定孩子
1. 注册时选择「我是家长」。
2. 登录后在绑定卡片输入**孩子的用户名**与**绑定码**（孩子在「我的」查看），点「绑定孩子」。
3. 可绑定多个孩子；多孩子时顶部出现孩子切换器。

### 2. 代孩子操作
- 切换到目标孩子后，可代其打卡、兑换、抽奖，查看其连续天数、积分、报告。
- 打卡/中奖会收到站内通知。

### 3. 解绑孩子
- 「👤 我的」→「📎 已绑定的孩子」→ 点对应孩子的「解绑」。解绑后可重新绑定其他孩子。

## 三、管理员操作指南

管理端地址：`http://<服务器>/admin/`，用管理员账号登录（默认 `admin`）。

### 1. 概览
- 首页查看用户数、打卡数、待审核数等统计。

### 2. 打卡审核
- 进入打卡列表，位置异常/人脸不通过的记录会高亮。
- 逐条「通过 / 拒绝」并可填写审核备注。**通过后**该打卡才计入有效、发放积分与抽奖资格。

### 3. 兑换审核
- 在兑换记录中查看学生兑换，进行发放/驳回处理。

### 4. 奖品管理
- 新增/编辑/删除奖品，设置分类、积分成本、库存、中奖概率、上下架。
- 抽奖券类奖品：勾选后设置每次兑换发放的券数量。

### 5. 闯关任务管理
- 新建闯关任务，设置奖励积分、开放条件/时间。
- 手动或定时开放任务；审核学生的闯关打卡。

## 四、常见问题（FAQ）

**Q：打卡一直提示失败/被拒绝？**
A：可能是人脸比对未通过（enforce 模式）。确保采集底图时光线充足、正脸清晰，打卡时本人出镜。若为演示环境无网络，联系管理员调整人脸策略。

**Q：照片上传提示无效？**
A：请上传真实拍摄的作业照片（体积 ≥ 5KB、清晰不模糊），避免纯色图或极小缩略图。

**Q：连续打卡断了怎么办？**
A：连续天数以「有效打卡的自然日」计算，断卡会重置当前连续天数（最长记录保留）。可在每月补卡额度内补卡。

**Q：家长看不到孩子数据？**
A：确认已用正确的用户名 + 绑定码绑定，且顶部切换到了该孩子。

**Q：登录提示太频繁（429）？**
A：触发了防暴力破解限流，请稍等一分钟再试。
